# Function Explanation
- 메모를 추가하면 ComposeViewController를 통해서 아래와 같이 메모가 화면에 나타나도록 만들었습니다.
- 메모가 화면에 보여질때 DetailViewController에서 DateFormatter를 작성해서 화면에 지정된 형식의 날짜와 시간이 표시되도록 만들었습니다.
